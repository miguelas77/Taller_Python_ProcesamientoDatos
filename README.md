## Descripción General
Este repositorio compila la serie completa de talleres prácticos desarrollados para el módulo de programación en Python. El objetivo principal fue dominar desde la sintaxis básica hasta la implementación de lógica algorítmica y programación orientada a objetos.



## Estructura del repositorio

El contenido está organizado secuencialmente para reflejar la curva de aprendizaje:

* **01_Cadenas/**: Manipulación de texto, indexación y slicing.
* **02_Tuplas/**: Estructuras inmutables y protección de datos.
* **03_Listas/**: Estructuras mutables y operaciones dinámicas.
* **04_Conjuntos/**: Teoría de conjuntos y eliminación de duplicados.
* **05_Diccionarios/**: Mapeo de datos clave-valor y optimización de búsquedas.
* **06_Condiciones/**: Lógica booleana y control de flujo (`if`, `else`, `elif`).
* **07_Bucles/**: Automatización de tareas repetitivas (`for`, `while`).
* **08_Funciones/**: Modularización y reutilización de código.
* **09_Clases/**: Introducción a la Programación Orientada a Objetos (POO).
* **10_Bono/**: Ejercicios de lógica algorítmica avanzada y resolución de problemas.
